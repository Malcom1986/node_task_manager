## Требования

* Node.js >= 14
* npm >= 6
* make >= 4

*Образец описания команд, необязательно именно так*

## Установка и запуск

* `make install` установка (с миграциями)
* `make build` сборка пакета
* `make start` запуск приложения

## Дополнительно

* `make lint` проверка линтером
* `make test` проверка тестами
